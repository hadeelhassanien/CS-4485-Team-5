# CreatorXp Application

### Full-stack project using:
- React (Frontend)
- Spring Boot (Backend)
- MySql (Database)

---

## Prerequisites

- Node.js
- npm
- Java
- Maven
- MySQL Server + MySQL Workbench

Check installed versions:

```
node -v
npm -v
java -version
javac -version
mvn -v
```
## Backend Setup
```
cd backend
./mvnw clean install
./mvnw spring-boot:run
```

## Frontend Setup
```
cd frontend
npm install
npm start
```
